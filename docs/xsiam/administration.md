### `xsiam_rbac_get_roles_v1`

Retrieve information about one or more roles created in your environment. Required license: **Cortex XSIAM Premium** or **Cortex XSIAM Enterprise** or **Cortex XSIAM Enterprise Plus**

**Parameters:**

- authorization (str): {api_key} (required)
- x_xdr_auth_id (str): {api_key_id} (required)
- request_data (Dict[str, Any]): A dictionary containing the API request fields. (optional)

**Returns:** List[types.TextContent]: OK
